## Run Locally

Clone the project

```bash
  git clone https://github.com/MarcoSolu/Recipes.git
```

Go to the project directory

```bash
  cd recipes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
